Tumblr uses OAuth 1.0a for authentication and communication. In order for users to authenticate with the Tumblr API, you must specify the ID, secret and redirect URI of your Tumblr app. You can create a new app at the following URL: https://www.tumblr.com/oauth/apps

Notice: When a user authenticates with the Tumblr API, an access token is saved along with information about the user. With the Tumblr API, access tokens will NOT expire.
